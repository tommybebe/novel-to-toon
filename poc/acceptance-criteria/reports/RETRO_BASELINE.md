# Retroactive Baseline Evaluation (Phase 5 only)

Generated by `poc/acceptance-criteria/scripts/run_evaluation.py`.

## Scope
- Only evaluates **final generated panels** in `phase5_generation`.
- Does **not** evaluate earlier phases.

## Evaluator mode
- `heuristic_offline` (no `ANTHROPIC_API_KEY` present in this environment)

## Outputs
- `evaluation_v0.1.0.json`
- `evaluation_v0.2.0.json`
- `comparison.json`

## What is evaluated
- Tier 1: technical validity (8 checks)
- Tier 2 (per panel): identity match, prompt adherence, style match, technical defects
- Tier 3 (per scene): character consistency, style consistency
- Tier 4 (per scene): sequential storytelling, emotional arc
